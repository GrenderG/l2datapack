Establishing a clan
: Enter "/establish" in the chat input window. In the current version, anyone can use this command.
Joining a clan
: Target the leader of the clan that you wish to join and enter "/join."
Viewing clan information
: Enter "/clan" in the chat input window. You must be a clan member to view this information.
Banishing a clan member
: The leader of the clan can banish a clan member from his own clan. Enter "/ban [character name]" in the chat input window.
Leaving a clan
: Enter "/leave" in the chat input window. Characters that are already members of a clan cannot create a new clan without first leaving their previous clan.
