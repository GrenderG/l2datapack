Talking
: Use the keyboard to input what you want to say. Chatting is possible even without pressing the Enter key.
Clan chatting
: You can also engage in chatting that is only within the same clan. The way to do this is to type @[what you want to say]. This function is not working completely yet.
Party chatting
: You can also engage in chatting that is only within the same party. The way to do this is to type #[what you want to say]. This function is not working completely yet.
Yelling
: You can send your message to a wider area. The way to do this is to type ![what you want to say]. This function is not working completely yet.
